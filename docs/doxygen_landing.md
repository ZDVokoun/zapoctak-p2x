# Residual arithmetics

You can find documentation of the interface of the library in these files:
  - @ref residue.h -- Operations with numbers in residue system
  - @ref conversion.h -- Functions which enable us to convert between the residue system and multi-precision representation, also includes functions for convertion between decimal and internal representations
  - @ref mp_number.h -- Operations with multi-precision representations
  - @ref config.h -- Default configuration used by the library (especially the moduli chosen)
